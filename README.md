Jeu d'arcade réalisé en avril 2018 en groupe de trois.

# Jouer sur Windows / Play on Windows
Exécutez / run:
> bin/SoCarreSDL.exe

# Jouer sur Linux / Play on Linux
Ouvrez un terminal dans le dossier racine / Open a terminal in the root folder
> ./bin/SoCarreSDL
